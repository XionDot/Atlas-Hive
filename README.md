## Installation

```
1. Download Atlas-1.0.0.dmg
2. Open DMG → Drag to Applications
3. Launch Atlas from Applications
4. Click the menu bar icon
```

> **Note:** Atlas is notarized by Apple - no Gatekeeper warnings.
